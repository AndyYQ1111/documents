## 1. Introduction

Welcome to **VaultKey Password Manager** ("VaultKey", "we", "our", or "us"). We are committed to protecting your privacy and personal information. This Privacy Policy explains how VaultKey handles information when you use the application.

VaultKey is designed as a **fully offline password manager**. We do not operate servers that collect or store your personal data.

By using the VaultKey application, you acknowledge that you have read and understood this Privacy Policy. If you do not agree with any part of this policy, please discontinue use of the application.

## 2. Core Principles

VaultKey is built on the following core privacy principles, in line with App Store and Google Play privacy requirements:

1. **Offline by Design**: VaultKey does not require an internet connection. All features operate locally on your device.
2. **User Data Ownership**: You retain full ownership and control of all data you choose to store in the app.
3. **Data Minimization**: We do not collect data beyond what is necessary for core functionality.
4. **Transparency**: We clearly disclose how data is handled within the app.
5. **Security First**: We use industry-standard security practices to protect locally stored data.

## 4. How We Use Your Information

### 4.1 Purposes of Use

All information processed by VaultKey is used solely to enable app functionality, including:

1. Managing and displaying stored credentials
2. Generating secure passwords
3. Encrypting and decrypting local data
4. Verifying user access via PIN or biometric authentication
5. Improving stability and performance **on the device only**

### 4.2 How Information Is Processed

* **On-Device Processing Only**: All encryption, decryption, and verification occur locally
* **No Data Sharing**: We do not share data with third parties
* **No Tracking or Profiling**: VaultKey does not track user behavior
* **No Advertising Use**: User data is never used for advertising or marketing

---

## 5. Information Storage

### 5.1 Storage Location

All your information is stored in secure storage areas on your device:

* **iOS Devices**: Using iOS Keychain and Secure Enclave
* **Android Devices**: Using Android Keystore and encrypted storage
* **Secure Storage**: All sensitive information is encrypted before storage

### 5.2 Storage Duration

Your information is stored for the following durations:

* While you use the application
* Removed immediately when you delete information
* Automatically deleted when you uninstall the app
* We do not retain any backup copies

### 5.3 Data Security

We implement multiple layers of security measures:

1. Device-Level Security: Utilizing operating system security features
2. Application-Level Security: PIN protection, biometric authentication
3. Data-Level Security: AES-256 encryption
4. Transmission Security: No data transmission occurs
